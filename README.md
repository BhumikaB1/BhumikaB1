## 🔨 Projects

**[Olist E-Commerce Revenue Intelligence Dashboard](https://github.com/BhumikaB1/Olist-PowerBI-Dashboard)**
Full BI pipeline on 100,000+ real Brazilian e-commerce transactions.
Built a star schema data model (5 tables), wrote DAX measures for MoM growth,
rolling 3M revenue, customer LTV, and repeat rate — then surfaced the insights
that matter: SP state drives 37% of revenue, repeat rate is only 3.01% (a clear
retention problem), and Black Friday 2017 was the single biggest acquisition spike.
Also implemented Row-Level Security for regional role-based access.
`Power BI` `DAX` `Power Query` `Star Schema` `RLS`

**[Gov Subsidy Navigator](https://bhumikab1.github.io/Government-subsidy-eligibility-calculator/)**
CivicTech platform using smart eligibility logic + Gemini AI to match citizens
to 30+ Maharashtra & Central government schemes. Built with data-driven filtering
and a conversational AI layer.

**[Insurance Premium Predictor](https://github.com/BhumikaB1/Insurance-premium-api)**
End-to-end ML pipeline — data preprocessing → model training → FastAPI backend
→ Dockerized deployment → Streamlit dashboard. Real predictions, live.

**[CureVeda](https://github.com/BhumikaB1/CureVeda_main)**
Cloud-based Ayurvedic clinic platform with custom EMR modules. Serverless backend,
cross-platform deployment, designed for accessibility.
